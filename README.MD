# CRUD - GO

API desenvolvida com GOLANG e MYSQL.
Funcionalidades.

- Inserção de um usuário.
- Listagem de usuários.
- Listagem de um usuário.
- Atualização de um usuário.
- Remoção de um usuário.

## Let's go run!

 - Necessário ter golang instalado em sua maquina, [Golang](https://golang.org/)。
 - Necessário MYSQL, [Mysql](https://www.mysql.com/)
 - Rode na raiz do proejto <go run index.go>.
